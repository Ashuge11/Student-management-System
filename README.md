# Student Management System

## Description
A Python-based Student Management System using OOP, NumPy, and Pandas.

## Features
- Add Student
- View Students
- Search Student
- Delete Student
- Rank Students
- Subject Topper
- Export CSV Report
- Analytics

## Technologies Used
- Python
- NumPy
- Pandas

## Concepts Used
- OOP
- Lists
- Dictionaries
- Loops
- CSV Handling

## Installation

pip install numpy pandas

## Run Project

python main.py
